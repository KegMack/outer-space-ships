INSTRUCTIONS

Movement: Click and Drag ship with mouse *or* use left and right arrows on keyboard

Start Level: 'spacebar'

Pause: 'p'

Quit to menu: 'q' when game is paused

Toggle Sound: 's'

ABOUT

This program was created using the following programs and languages

  • JavaScript
  • JQuery
  • JqueryUI
  • CSS3
  • HTML5
  • Sublime text editor

It is meant to be a throwback to past-generation games such as Arkanoid, by Taito, and Breakout, by Atari.

It has been a goal of mine as long as I can remember to create a functional video game, and this was a great opportunity to accomplish that, while also learning the above languages.
